# Check that the sample is bigger than the mincellcount

Check that the sample is bigger than the mincellcount

## Usage

``` r
checkSampleMinCellCount(sampleSize, minCellCount, messageStore)
```

## Arguments

- sampleSize:

  sample size for sampling

- minCellCount:

  minimum cell count below which to obsure results

- messageStore:
